本地 clone 地址：

```
$ git clone git@github.com:yybeaut/yybeaut.github.io.git
```

在 themes 文件夹下复制 maupassant
`$ npm i`

#### 基本维护命令

```bash
$ npm install hexo-deployer-git --save
$ hexo c
$ hexo g
$ hexo d
$ hexo clean && hexo g && hexo d
```

### hexo server

```bash
$ hexo server
```

### Generate static files

```bash
$ hexo generate
$ hexo s
```

### Deploy to remote sites

```bash
$ hexo deploy
```
